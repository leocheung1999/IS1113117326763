Platform as a Service (PaaS) - Cloud 9



Platform as a service (PaaS) used for applications, and other development, while providing cloud components to software.
provides a platform and environment to allow developers to build applications and services over the internet.


PaaS services are hosted in the cloud and accessed by users simply via their web browser.

How it works ?

  • Creation of software applications
  • Management and Support
  • Pay-Per-Use
  • Automatic Upgrades

Examples of PaaS

  • Cloud 9
  • Heroku
  • Microsoft Azure
  • Apprenda










Benefits of PaaS

  1. Don't have to invest in physical infrastructure
  2. Flexibility
  3. Adaptability
  4. Teams in various locations can work together
  5. Security
