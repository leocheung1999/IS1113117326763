Software as a Service (SaaS) - Salesforce


Software as a service (SaaS) uses the web to deliver applications that are managed by a third-party vendor and whose interface is accessed on the clients' side.
Most SaaS applications can be run directly from a web browser without any downloads or installations required.

SaaS makes it easy for business' to stream their maintenance and support.

Examples of where SaaS can be used by enterprises:

  • Accounting and invoicing
  • Tracking sales
  • Performance monitoring

Examples of SaaS

  • Salesforce
  • Office 365
  • Google Apps
  • Netflix









Benefits of SaaS

  1. No hardware costs
  2. Usage is scalable
  3. Accesible from any location
  4. No initial set up costs
  5. Automatic Updates

Platform as a service (PaaS) used for applications, and other development, while providing cloud components to software.

Infrastructure as a Service (IaaS) - Amazon Web Services

Infrastructure as a service (IaaS) are self-service models for accessing, monitoring, and managing remote datacenter infrastructures, such as , storage, networking, and networking services.
IaaS users are responsible for managing applications, data, runtime, middleware, and OSes. Providers still manage virtualization, servers, hard drives, storage, and networking.

The hardware is provided by an external provider and managed for you.

How it works ?

  • The Cloud
  • Virtualised Hardware
  • Cloud Servers

Examples of IaaS

  • Amazon Web Services
  • Cisco Metapod
  • Microsoft Azure
  • Google Compute Engine (GCE)










Benefits of IaaS

  1. Scalability
  2. No hardware CAPEX
  3. Location independence
  4. Physical security of data centre location
  5. No single point of failure

Communication as a Service (CaaS) - Gmail

Communication as a service (CaaS) is an outsourced enterprise communications solution that can be leased from a single vendor.
The CaaS vendor is responsible for all hardware and software management and offers guaranteed Quality of Service (QoS).

CaaS is an alternative to the model of buying and installing licensed software on site.

How it works ?

  • Services over the internet
  • Business users just sign in

3 different types:

  • Voice over Internet Protocol (VoIP)
  • Internet Telephone Solutions
  • Video Conferencing Services

Examples of CaaS

  • Gmail
  • Skype
  • FaceTime
  • Facebook Messenger








Benefits of CaaS

  1. Scalability

  3. Self-Service
  4. Global Accessibility
  5. Expandabilty
